# RelinkFSMToolkit

WIP FSM/Quest Editor for Granblue Fantasy: Relink. MVVM + MVVM Toolkit + DI + AvalonDock. 

## State

* Can load & view multiple FSMs
* Can load FSM list from quest
* Can add/remove components
* Can save individual FSM graphs
